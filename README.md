Télécharge tes playlists YouTube sur PC et écoute-les hors-ligne sur Android.

| Plateforme | Lien |
|---|---|

| 📱 **Android** (app mobile) | [app-release.apk](https://github.com/Yma061/playlist-mp3-downloader/releases/download/v2.0.0/app-release.apk) |

---

## 🔄 Comment ça marche ?

1. Lance **PlaylistManager.exe** sur ton PC
2. Télécharge une playlist YouTube via l'app (YouTube → MP3)
3. Clique sur **📱 Télécharger sur mobile** → l'interface web s'ouvre sur `http://localhost:8888`
4. Connecte ton téléphone et ton PC au **même Wi-Fi**
5. Ouvre l'app Android → **Sync bureau** → **Détecter automatiquement**
6. Sélectionne ta playlist → **Synchroniser**
7. Clique sur la playlist → navigue dans les titres → écoute **hors-ligne** 🎧

---

## 📱 App mobile

- Bibliothèque avec toutes tes playlists synchronisées
- Interface style Spotify : clique sur une playlist pour voir les titres
- Lecteur plein écran avec slider de progression, prev/pause/next
- Détection automatique du PC via Wi-Fi (mDNS)
- Lecture **100% hors-ligne** une fois synchronisé

---

## ⚠️ Prérequis

- **Windows 10/11** — aucune installation requise, double-cliquez pour lancer
- **Android 8.0** ou supérieur

---

## 📦 Installation Android

1. Télécharge le fichier `.apk`
2. Sur ton téléphone : **Paramètres → Sécurité → Autoriser les sources inconnues**
3. Ouvre le fichier APK et installe
