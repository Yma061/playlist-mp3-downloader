# Playlist Manager — Android

Écoute hors-ligne les playlists synchronisées depuis le serveur desktop.

## Téléchargement

Dernière version : **v2.0.0**  
**[⬇ Télécharger l'APK](https://github.com/Yma061/playlist-mp3-downloader/releases/download/v2.0.0/app-release.apk)**

> Android 8.0 ou supérieur requis.

### Installation

1. Télécharge le fichier `.apk`
2. Sur ton téléphone : **Paramètres → Sécurité → Autoriser les sources inconnues**
3. Ouvre le fichier APK et installe

---

## Fonctionnalités

- Bibliothèque musicale locale
- Sync Wi-Fi automatique avec le serveur desktop (détection mDNS ou IP manuelle)
- Interface style Spotify : clique sur une playlist pour voir tous les titres
- Lecteur plein écran avec slider de progression, boutons prev/pause/next
- Mini-player persistant en bas de l'écran, cliquable pour ouvrir le lecteur
- Lecture **100% hors-ligne** une fois synchronisé
- Thème clair / sombre

---

## Utilisation

1. Lance **PlaylistManager.exe** sur ton PC ([télécharger](https://github.com/Yma061/playlist-mp3-downloader/releases/download/v2.0.0/PlaylistManager.exe))
2. Télécharge une playlist YouTube depuis l'app desktop (YouTube → MP3)
3. Connecte ton téléphone et ton PC au **même Wi-Fi**
4. Ouvre l'app → **Sync bureau** → **Détecter automatiquement**
5. Sélectionne ta playlist → **Synchroniser**
6. Clique sur la playlist → choisis un titre → écoute hors-ligne 🎧
